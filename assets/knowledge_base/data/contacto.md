# Contacto MEDIF ESTRUCTURAS
- **Correo para cotizaciones estructurales e hidrosanitarias:** `eduardo.mediavilla@medifestructuras.com`. Envíanos planos, ubicación, tipo de estructura, niveles, plazos y restricciones para recibir respuesta con los próximos pasos.
- **Correo administrativo y cursos (inscripciones, pagos, soporte general):** `eduardo.mediavilla@medifestructuras.com`.
- **WhatsApp:** +357 96 863 257 (uso preferente para seguimiento de inscripciones o aclaraciones rápidas).
- **Sitio web principal:** https://medifestructuras.com, desde donde se registran cursos, se descargan recursos y se accede a la plataforma de formación.

## ¿Qué incluye cada canal?
- `eduardo.mediavilla@...`: solo para diseño, cálculo o cotización de proyectos estructurales o hidrosanitarios; es el único canal para ese tipo de respuesta.
- `inscripciones@...`: cursos, asesorías técnicas, servicios digitales, facturación y administración.
- WhatsApp: confirmación de agenda, recordatorios y seguimiento ligero (no para enviar planos largos).
- También puedes hacer clic en “Enviar mis datos” desde el chat para que el equipo reciba tu número, la mejor franja horaria y te llame de vuelta con propuestas concretas, fechas o el link de pago correspondiente.
