# Overview software
Medif Estructuras trabaja con CYPE (CYPECAD, CYPE 3D, Strubim), SAP2000, ETABS y herramientas BIM para cálculo, análisis dinámico y coordinación de instalaciones.

## Qué cubre
- Modelado y cálculo estructural e hidrosanitario en una misma suite.
- Análisis no lineal, puentes grúa y puentes de hormigón con SAP2000/ETABS.
- Coordinación BIM con Open BIM Layout y CYPE Cost Estimator.
